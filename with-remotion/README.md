## Code Hike + Remotion

Blog post: https://v1.codehike.org/blog/remotion

Code Hike docs: https://v1.codehike.org/docs/
Remotion docs: https://www.remotion.dev/docs/

### Clone this example

```bash
$ npx degit code-hike/examples/with-remotion my-video
$ cd my-video
$ npm install
$ npm run start
```
